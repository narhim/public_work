This directory contains the evaluations scores (precision, f1 score and number of documents with at least one relevant document) for the document information retrieval systems contained on the sister directory source_code.

-results_basic_tf-idf.txt contains the results for the basic tf-idf implementation.
-results_rerank_bm_k_value(...).txt contain the results for the bm implementation for reranking with different k values and the same b (0.1).
